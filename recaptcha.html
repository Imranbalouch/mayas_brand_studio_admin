What you need to make reCaptcha work?

To enable reCaptcha, you must obtain an API key from the Google reCaptcha website and configure it in your application settings.

  • Create a new project in the Google Cloud Console.
  • Go to the reCAPTCHA Admin Page and register a new site.
  • Copy the reCAPTCHA Site Key and Secret Key provided after registration.
  • Go to your application settings and enter the API key and secret key.
  • To integrate reCAPTCHA into a form, add the google_recaptcha class before the submit button tag.
